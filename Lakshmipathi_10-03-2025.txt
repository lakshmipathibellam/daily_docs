--> Press enter

--> now open the GitHub account go to settings and open ssh key 

--> Right side showing new ssh key click on it it will ask enter the title. give the title name and come down

HOW TO INITIALIZE GIT AND PUSH CODE IN VSCODE :

--> Open vscode and open the folder

--> Take on new file and write some code

--> Then go to terminal open new terminal 

--> Enter the commands :

--> git init

--> git add file name 

--> git commit -m "first commit"

--> git branch -M main

--> git remote add origin git@github.com:lakshmipathibellam/sample.git

--> git push -u origin main

HOW CREATE BRANCH AND SWITCH TO ONE BRANCH TO ANOTHER BRANCH :

--> Create branch : git switch -c branch name (or) git checkout -b branch name

--> Switch Back to the Main Branch : git switch main (or) git checkout main 

--> If you want to see branch : git branch
